#无监督学习 
--
不输入正确答案让机器自己学习找规律，进行分类，即只是输入数据x但是不包含标签y，但是机器会发现一些相同的内容并且将其聚类划分
例如：聚类算法——机器进行相关的分类，划分标签--将一些内容划分到一个组里（由数据特征发现）

回归模型的训练：
对于损失函数的定义，cost function——（又叫平方代价误差函数）残差平方和/总数*2（计算一个相对的拟合），我们要做的就是拟合最小的损失函数，然后得到这个时候的参数孕激素和我们训练出来的最佳参数
研究最小的损失函数，使用梯度下降方法来取得最小的参数（因为多参数下的损失函数J是构成类似等高线的形式）——我们要找到梯度的最低点，也就是拟合最好的情况，然后找到这个情况下的参数，就是模型的参数。

---


关于梯度下降，其适用于不只是处理两个参数，其可以处理对多个参数下的损失函数模型。寻找梯度下降的最低点，你们我们要找到下降速度最快的一侧，也就是要对这个空间求偏导但是也可能导致你产生幻觉，就是它引导你能到达的最小值为局部最小值（local minimum）
梯度下降的数学原理:
	对权重w的处理
$$
w = w -α*d/dw *J(w,b)
$$
***这里的α是学习率（也就是梯度下降的步长），这里dy/dx就是成本函数的导数（更准确的说这个是偏导），但是这个J（w，b）有两参数，只是对w求导***
	对于bais的计算：
	
$$
	b = b - α * d/db * J(w,b)
$$
这两步对w，b的update的过程是同时进行的，（同步更新）也就是说，这一个w和b使用的损失函数j是同一个，而不会因为某一个先更新导致J改变![[../编程以及AI/image/懒得处理的一些img/Pasted image 20260131134514.png]]
学习率是一个正数，选择较小的学习率会导致步长小，计算时间长，选择较大的学习率会导致跨过最低点minimum。这个公式里面，J和W都是相互影响的，也就是说我们需要对于每一个点都计算（由于有学习率的存在其实本质上的点是离散的）
==在单一数据输入的·时候，你输入中我们写为ndarray的形式来输入样本特征，是由于对应的相当于二维空间内部的向量数据输入==
```python
# 房价预测  
import numpy as np  
from sklearn.linear_model import LinearRegression  
x = np.array([121,125,131,141,152,161]).reshape(-1,1)  
y = np.array([300,350,425,405,496,517]).reshape(-1,1)  
# print(x.shape,y.shape))  
# 2、定义线性回归模型  
lr  =LinearRegression()  
# 3、模型训练  
lr.fit(x,y)  
# 4、模型评估  
w = lr.coef_#权重  
b = lr.intercept_#偏置值  
print(f"模型是{w}*x+{b}")  
  
x_test = np.array([[130]])  
y_pred = lr.predict(x_test)#对于y的预测  
print(f"房价预测结果为：{y_pred}"
```

因此，其具体的线性回归的的参数更新可以写为
$$
w = 1/m · sum((f(xi) - yi)^2) ·xi

#特征工程
--
处理拟合效果更佳的梯度下降，使用特征缩放来处理数据，减少训练量 ——how：
		特征的缩放：归一化处理，有多种归一化。（对于那一些数据横跨范围大的数据，需要使用归一化形式，降低偏差）
		好的，根据你屏幕上的内容，我为你总结了数据归一化（更准确地说是Z-Score标准化）的处理流程和代码实例。

	数据归一化（Z-Score标准化）流程
	
![[../编程以及AI/image/懒得处理的一些img/Pasted image 20260305172419.png]]
	
	这种归一化方法的流程是，对数据集中的每个特征（列）进行独立的处理，具体步骤如下：
	
	1. **减去均值 (Subtract the mean)**：用该列的每个数据点减去该列所有数据的平均值。这一步会将数据的中心点移动到0。
	2. **除以标准差 (Divide by the standard deviation)**：将上一步得到的结果，除以该列数据的标准差。这一步会将数据的分布“拉伸”或“压缩”，使其标准差为1
```
		# 对data2数据框中的所有数值列进行归一化（Z-score标准化）
		 每个值 = (原值 - 该列的均值) / 该列的标准差 
		 data = (data - data.mean()) / data.std()
		 #使用pandas库用于处理数据
```
对于梯度下降的检测，可以使用判断是否函数收敛（利用plot.plt方法），观察在某一个区间范围内的J损失函数的减小值是不是小于sigmoid值。利用图像，可以对于损失函数的图像进行判断（还可以判断有无代码写错了，或者是学习率alpha过大从而越过最低点导致J上升，也就是J损失函数不是稳定下降的，可以使用图像判断）

# 逻辑回归

使用sigmoid函数激活
![[../编程以及AI/python/assets/Matplotlib/image-1.png|348x347]]
$$
g(z) = \frac{1}{1 + e^{-z}}
$$
sigmoid函数是激活函数利用对于z的处理，然后对其进行分类
什么是决策边界就是从数学角度上看，可以激活sigmoid函数的原函数内部多个特征下的下的取值可能
对于逻辑回归中 ，单个变量的损失函数是
```python
loss=0
f(w,b)=sigmoid(x[i]*w+y[i])
loss=f(w,b)-y[i] 
```
成本函数的梯度下降，其实和对应的linear很相似但是这里是f_wb,是由sigmoid函数复合而成的
#逻辑回归的损失函数$L = -[y \log(a) + (1-y) \log(1-a)]$ !

## 关于决策边界
就是分别区分不同特征的分界线 ，如在逻辑回归的函数内部就是一条直线区分多个特征（就是标签），对于分类的决策边界就是处理多标签的问题，在边界一侧的是应该标签，在另一侧的为另一个标签![[assets/机器学习Machine learning入门/file-20260407102550435.png|457|321x382]]

## 使用regularization
降低各个参数的大小，但是做到了保留所有的特征features。一般正规化处理都是参数w，因为对bais处理效果不明显
![[assets/机器学习Machine learning入门/file-20260407102550436.png|534]]

对于正则化的成本函数的改变是增加了一项正则项（对于 线性回归默写和逻辑回归的都是添加一个正则项！）![[assets/机器学习Machine learning入门/file-20260407102550442.png]]
对于正则化的作用就是利用多增加的这一项来控制参数w的改变的变化，因为对于梯度下降总是要找到最低点使得函数拟合最好，所以当lambda增加的时候对应的其他参数就会相应的降低
在对其进行梯度下降的时候，会将lambda的内容一起算上，在处理梯度下降的时候下降的速度变缓了![[assets/机器学习Machine learning入门/file-20260407102550437.png|615]]

根据实验结果来看，对于参数lambda地改变尽可能地。小其对应的拟合效果会更加贴近更好的模型，同时增加w对应的维度也可以使得效果更好，对于分类问题正则化的作用更加明显，可以极大程度改变模型的泛化能力

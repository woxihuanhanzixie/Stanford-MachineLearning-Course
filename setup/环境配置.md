# 环境配置指南 / Environment Setup Guide

本项目推荐使用 **Anaconda** 或 **Miniconda** 进行 Python 环境管理。

---

##  第一步：安装 Anaconda/Miniconda

1. 前往 [Anaconda 官网](https://www.anaconda.com/download) 或 [Miniconda 官网](https://docs.conda.io/en/latest/miniconda.html) 下载适合你操作系统的安装包。
2. 按照默认提示完成安装。

##  第二步：创建虚拟环境

打开终端（Windows 用户请使用 `Anaconda Prompt`），执行以下命令创建一个名为 `ml-ng` 的环境：

```bash
conda create -n ml-ng python=3.10 -y
```

##  第三步：激活并安装依赖

1. **激活环境**:
   ```bash
   conda activate ml-ng
   ```

2. **安装项目依赖**:
   项目根目录下的 `setup/requirements.txt` 包含了运行所有实验所需的库。
   ```bash
   pip install -r setup/requirements.txt
   ```

##  第四步：使用 Jupyter Notebook (VS Code)

推荐在 **VS Code** 中运行项目中的 `.ipynb` 文件。

### 1. 安装必要插件
在 VS Code 扩展商店中搜索并安装：
- **Python** (Microsoft)
- **Jupyter** (Microsoft)

### 2. 配置 Jupyter 内核 (Kernel)
1. 打开 `labs/` 目录下的任意 `.ipynb` 文件。
2. 点击编辑器右上角的 **"Select Kernel"**。
3. 选择 **"Python Environments..."**。
4. 在列表中选择你刚才创建的 **`ml-ng`** 环境。

### 3. 安装 ipykernel (如提示)
首次运行代码块时，如果提示 "Running cells requires ipykernel package"，请点击 **"Install"** 即可。

---

##  常见问题

- **依赖安装失败？** 尝试更换 pip 源（如使用清华源）：
  ```bash
  pip install -r setup/requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```
- **找不到环境？** 重启 VS Code 或在命令行运行 `conda init` 后重新打开。

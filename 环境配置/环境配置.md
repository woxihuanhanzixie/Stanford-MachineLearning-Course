本项目推荐使用 Anaconda 进行环境管理。
## 第一步：安装Anaconda


官方网址：[Download Anaconda Distribution | Anaconda](https://www.anaconda.com/products/distribution#download-section)


下载并安装，按照默认setting来就好。

  

## 第二步：配置环境


在Windows搜索栏下搜索`Anaconda Prompt`,打开后，你会看到前面有 `(base)` 字样，说明进入该环境内部。



在加载完成后，我们开始配置一个ml-ng的环境。

  

继续输入以下命令：


```

conda create -n ml-ng python=3.10

```

  

## 第三步：激活你要用的环境

`conda activate ml-ng`

你会看到命令行前面变成：

`(ml-ng) C:\Users\...`

 **看到 `(study)`，才算“进入这个环境”**



## 第四步：安装你要的包

比如：

`conda install numpy`

或：

`pip install torch`

  
这一部分库的目录我放在requirement.txt中，可以用ai查怎么把它们一次性全部安装下来。如果还没有安装pip，建议先快速安装一个pip（非常有用！）。安装第三方库时，如果pip install不行可以尝试使用conda install，两个总有一个能行。

## 第五步：使用jupyter notebook
==这里使用vscode==
### 1. 安装 VS Code 插件

首先，确保你的 VS Code 已经安装了以下插件：

- **Python** (Microsoft 官方出品)
    
- **Jupyter** (Microsoft 官方出品)


### 2. 选择 Jupyter 内核 (Select Kernel)

这是最容易出错的一步。进入 VS Code 后，打开你的 `.ipynb` 文件，按照以下流程操作：

1. **点击右上角**：在 Notebook 界面右上角，你会看到类似 `Select Kernel` 或当前 Python 版本的文字。
    
2. **选择 Python 环境**：在弹出的搜索框中，选择 **"Python Environments..."**。
    
3. **找到 ml-ng**：从列表中找到路径包含 `anaconda3/envs/ml-ng` 的项。
    
    - _提示：如果没有看到，点击右上角的“刷新”按钮。_
